RIT Program

Offered by SSE in cooperation with the University of Technology in Vienna, Austria, and RISE (Research Industrial Systems Engineering), RIT stands for RISE institute of Higher Technologies Program.

Objective

The overall goal of the RIT Program is to transfer additional knowledge about hands-on computer programming and software engineering to CSE students Sanskrithi. There is a strong focus on practical assignments, where students have to solve problems by writing code. This should help them in practicing computer programming and to develop problem solving skills. It should also promote students to teach themselves various topics in software engineering using free online resources.

Duration of the Program

The total number of sessions/hours depends on the academic calendar and available resources, ideal time is three hours per week. The numbers of weeks depend on the module.

Each class session consists of answering questions and doubts of the students regarding the taught topics and/or current assignments. There will also be an interactive live coding session. Students have time to work on their assignments and get their queries answered. The solutions to the assignments will be submitted using the online learning platform Moodle and for some assignments, students must participate in a personal student-teacher submission review.

Recommended websites and books vary from module to module and will be provided on the online learning platform Moodle. Since technology evolves very quickly, we try to adapt the courses/modules from semester to semester with current topics and resources to learn taught topics. Resources for learning will be also shared from time to time with the students via the online messenger for communication throughout all modules. Websites/Resources used for learning programming are mentioned in each module.

MODULES

Module 1

In this module, students take to setting up necessary tools and learning skills that are needed in every module like Slack, Emails (writing & using), typing practice, and resources for self-learning. A visual programming language (Scratch) will be used to develop the mindset and thinking as a computer programmer. It includes 7 teaching-blocks á 2 weeks each.

Topics, Tools & teaching methods

  • Slack for communication
  • Practice typing with Typing Club
  • Reflect using a diary & submission reviews
  • Programming with Scratch

Content Flow

  • Email → Write & Use
  • Stackoverflow → Explanation + Use
  • Open Source → Videos + Questions (audio, video, doing)
  • Open Participation → Definition + Example OSM + active participation (audio, video, doing)

Promote self-learning

  • Provide resources where to find / to learn
  • Apply newly gained knowledge

Module 2

In this module, students will learn the basics of computer programming. There will be many small assignments to solve using Java. It includes 6 teaching-blocks à 2 weeks each.

Topics, Tools & teaching methods

  • Programming → Java
  • Reflect → Diary

Content Flow

  • Command Line / Bash → Read + Use
  • Compiler → Hello World + Use bash
  • Java
    • Teach fundamentals by programming simple games
    • Teach fundamentals by solving small riddles (problems)
    • Add features every assignment

Promote self-learning

  • Sololearn for Java

Module 3

In this module, students continue practicing computer programming using Java. The assignments will contain a larger workload compared to previous modules, but so is the time available to develop a solution. During 3 teaching-blocks à 4 weeks each, students will implement their first bigger Java application. After each teaching-block students will have to participate in a submission review about their submitted solutions.

Topics, Tools & teaching methods

Programming

  • Object-Oriented Programming in Java
  • HTML, CSS, and JavaScript

Reflect

  • Students will have to write a diary where they reflect on their gained knowledge and on the problems, they faced during creating solutions
  • The diary will be created using HTML, CSS, and JavaScript

Content Flow

  • Diary
    • Use HTML for structuring the content
    • Use CSS for styling
    • Use JavaScript for making the diary interactive
  • Java
    • Principles of Object-Oriented Programming
    • Writing readable code
    • Basics of computer programming

Promote self-learning

  • KhanAcademy for HTML, CSS, and JavaScript
  • Sololearn for Java

Module 4

In module 4 students deepen their knowledge in computer programming with Java. They will continue using Object-Oriented Programming and will also focus on Software Architecture. Students will learn about various design patterns and how to structure their Java code.

Additionally, they will use an SQLite database for reading and writing data. Another essential skill that students will learn is how to use and understand Git and GitHub. This module includes 5 teaching-blocks à 3 weeks each.

Topics, Tools & teaching methods

  • Programming
  • Software Architecture
  • How to use an SQLite database
  • HTML, and CSS for the diary
  • git and GitHub

Reflect

  • Students will have to write a diary where they reflect on their gained knowledge and also on the stopblocks, they faced during creating solutions
  • The diary will be created using HTML, CSS, and JavaScript

Content Flow

  • Diary
    • Use HTML for structuring the content
    • Use CSS for styling
  • Java
    • Principles of Object-Oriented Programming
    • Writing readable code
    • Software Architecture
    • Basics of using an SQLite database for reading and writing data

Promote self-learning

  • KhanAcademy for SQL
  • Sololearn for Java
